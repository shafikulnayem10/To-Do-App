# ✅ To-Do App

A clean, modern, and fully responsive To-Do application built with vanilla HTML, CSS, and JavaScript. Features a sleek glassmorphism UI with smooth animations, local storage persistence, and a polished dark theme.

---

## 🔗 Live Demo

> **[Click here to view the live site](https://to-do-app-lilac-pi.vercel.app/)**  






---

## ✨ Features

- ➕ Add new tasks instantly
- 🗑️ Delete tasks with smooth fade-out animation
- 💾 Persistent storage using **localStorage**
- 📱 Fully **responsive** — works on all screen sizes
- 🎨 Glassmorphism dark UI with animated background orbs
- 🔢 Live task counter
- ⚡ Zero dependencies — pure HTML, CSS & JS

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure & Semantics |
| CSS3 | Styling, Animations, Responsive Design |
| JavaScript (ES6+) | Logic, DOM Manipulation, localStorage |
| Font Awesome 6 | Icons |
| Google Fonts | Syne + DM Sans typography |

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/todo-app.git
```

### 2. Navigate into the project folder

```bash
cd todo-app
```

### 3. Open in browser

Simply open `index.html` in your browser — no build step or server required.

```bash
# Or with VS Code Live Server
code .
```
